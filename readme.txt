The most simple post like & dislike plugin for WordPress. 

== Description ==

This plugin is a simple utility that helps you adding widgets to the widget areas when there are too many of them. It sticks the widget areas to the right. 
This plugin is inspired by [Admin Sticky Widgets](https://wordpress.org/plugins/admin-sticky-widget-areas/) 
But instead of CSS it uses javascript library [Theia Sticky Sidebar](https://github.com/WeCodePixels/theia-sticky-sidebar)
